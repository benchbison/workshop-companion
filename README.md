# Workshop Companion

A woodworker's field companion, terms, tools, wood species, joinery, finishing, and shop knowledge. Built to be useful at the bench, not impressive on a screen.

**Live app:** [companion.benchandbison.com](https://companion.benchandbison.com)

---

## What it is

A single-file Progressive Web App covering the reference material a working woodworker reaches for most. Organized, searchable, and fully usable without an internet connection after the first visit.

**What's inside:**
- 15 woodworking terms with definitions and cross-links
- 23 hand and power tools with technique notes
- 30 wood species, domestic, European, and exotic, with Janka hardness, workability, finishing notes, and food safety
- Joinery types, clamps, workbench designs, hardware and fasteners
- Wood finishing, finishing schedules, gluing, milling, layout, sharpening
- Common mistakes, finish troubleshooting, wood defects
- 7-step hand-cut dovetail walkthrough
- Joint decision guide (interactive)
- Board foot calculator and wood movement calculator
- Shop safety, maintenance, drying, setup references
- Beginner guide and starter projects
- Workshop journal, timestamped entries, auto-save, export

---

## Technical

- Single HTML file, all data inline as JavaScript arrays
- No framework, no build step, no dependencies beyond Google Fonts
- Progressive Web App, installable to home screen on iOS and Android
- Service worker for full offline use after first visit
- localStorage for workshop journal (device-local, no server)
- Full-text search with scoring across all sections

---

## Install to your phone

Visit [companion.benchandbison.com](https://companion.benchandbison.com) in Safari (iOS) or Chrome (Android), tap Share, and choose **Add to Home Screen**. Opens full-screen like a native app and works offline.
